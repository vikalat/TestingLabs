№1  Тестирование регистрации
    1. Переходим на https://deal.by/
    2. Нажимаем кнопку "Зарегистрироваться" на верхней панели 
    3. Выбираем "Как покупатель" в выпадающем окне
    4. На открывшейся странице выбираем "социальные сети" и "Вход через Google"
    
    Ожидаемый результат: Получение доступа к личному кабинету

№2  Тестирование строки поиска товара
    1. Переходим на https://deal.by/
    2. Вводим в строку поиска товара "еда"
    3. Нажимаем клавишу Enter
    
    Ожидаемый результат: произойдет перенаправление на список ссылок относящийся к слову еда

№3  Тестирование фильтрации услуг
    1. Переходим https://deal.by/
    2. Вводим в строку поиска товара "еда"
    3. Нажимаем клавишу Enter
    4. На открывшейся странице в списке выбираем "Ланч боксы"

    Ожидаемый результат: на странице будут присутствовать только ланч боксы

№4  Тестирование подачи частного объявления
    1. Переходим на https://deal.by/ и выполняем вход в свой аккаунт
    2. Нажимаем кнопку "Подать частное объявление" на верхней панели
    3. В открывшейся странице загружаем изображения, заполняем данные и нажимаем кнопку "Сохранить и опубликовать"
    4. Во всплывающем окне вводим код из сообщения, пришедшего на указанный телефон
    5. Нажимаем кнопку "Создать"
    
    Ожидаемый результат: открытие страницы с объявлением, которое только что создали

№5  Тестирование входящих сообщений.
    1. Переходим на https://deal.by/
    2. Выполняем вход в свой аккаунт
    3. Наводим курсор на имя пользователя
    4. В выпадающем окне выбираем "Сообщения"
 
    Ожидаемый результат: страница со входящими сообщениями

№6  Тестирвоание удаления своего частного объявления
    1. Переходим на https://deal.by/
    2. Выполняем вход в свой аккаунт
    3. Наводим курсор на имя пользователя
    4. В выпадающем окне выбираем "Мои частные объявления"
    5. Нажимает на многоточие напротив объявления
    6. В выпадающем окне выбираем "Удалить"
    
    Ожидаемый результат: изменён статус объявления

№7  Покупка товара
    1. Переходим на https://deal.by/
    2. Выполняем вход в свой аккаунт
    4. Открываем интересующий нас товар
    5. Нажимаем на кнопку "Купить"
    6. В открывшемся окне нажимаем "Оформить заказ"
    7. На странице оформления заполняем необходимые поля данными нажимаем "Оформить заказ"

    Ожидаемый результат: Новое окно с информацией о заказе

№8  Сохранение товара в "Избранном".
    1. Переходим на https://deal.by/
    2. Выполняем вход в свой аккаунт
    3. Открываем интересующий нас товар
    4. Нажимаем на изображение сердечка
    5. Нажимаем на кнопку "Избранное" (изображение сердечка) под именем пользователя

    Ожидаемый результат: страница с сохранёнными товарами
    
№9 Тестирование чата с продавцом
    1. Переходим на https://deal.by/
    2. Выполняем вход в свой аккаунт
    3. Открываем интересующий нас товар
    4. Нажимаем кнопку "Написать"
    5. В открывшейся форме пришем сообщение и отправляем
    
    Ожидаемый результат: возможность связаться с продавцом 

№10 Тестирование написания отзыва
    1. Переходим на https://deal.by/
    2. Выполняем вход в свой аккаунт
    3. Выбираем необходимый товар
    4. Нажимаем кнопку "Добавить отзыв о товаре"
    5. На открывшейся странице пишем отзыв
    7. Нажимаем кнопку "Добавить отзыв"
    
    Ожидаемый результат: Отзыв сохранится
